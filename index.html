Dans les deux propositions javascript autour des recettes : "Les petits plats", l'ensemble de 3 structures itératives autour de la recherche générale des recettes diffèrent.
Dans la version 1 de l'algorithme, les 3 structures ont recours aux méthodes iteratives liées aux tableaux (Array). La premiere structure utilise une fonction JS .Filter tandis que les deux dernières sont des fonctions JS native .Foreach.
Dans la version 2 de l'algorithme, les 3 structures ont recours à la boucle For, native de JS. Une variable d'iteration se charge de controler les tours de la boucle et répète les instructions suivant les conditions.
